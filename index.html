Виды селекторов
Селектор Описание
* Выбор всех элементов
#myId Выбирает элемент с идентификатором myId
.myClass Выбирает все элементы с классом myClass
тег tg Выбор всех тегов td
:hover Выбор всех псевдоклассов hover
[type] Выбирает все элементы с атрибутом type
Совмещенные
теги span div Задание правил для всех тегов span и div
.class .class Задание правил для всех указанных class
тег + тег Выбирает все элементы, которые расположены рядом
.class>.class Выбор элементов по вложенности
